To start using grid simply add a display grid to the container element and use grid-template-columns to create your columns:

                    
    div:first-of-type ul {
        display: grid;
        grid-template-columns: 1fr 10em 1fr;
        gap: 1em;
    }

Using span you can span a grid element across multiple columns or rows:

                    
    div:first-of-type ul li:nth-child(2) {
        background-color: goldenrod;
        grid-column: span 2;
    }               
                
                    
    div:first-of-type ul li:nth-child(5) {
        background-color: hotpink;
        grid-row: span 2;
    }

The first thing you need to do is declare which grid elements you want to use. To do this, add a grid-area to each grid element. You can give it a name like item1, item2 etc.

                    
    div:nth-of-type(2) li:nth-of-type(1) {
        grid-area: item1;
    }

    div:nth-of-type(2) li:nth-of-type(2) {
        grid-area: item2;
    }

    /* etc etc... */

Now you can use grid-template-areas on the container to create the layout you want. Make sure your columns and rows are setup correctly so it doesn't break.

                    
    div:nth-of-type(2) ul {
        display: grid;
        grid-template-columns: 20em 1fr 1fr;
        grid-template-rows: 20em 1fr 20em;
        grid-template-areas:
            'item6 item2 item2'
            'item3 item2 item2'
            '.     item4 .'
            'item1 item1 item5';
        gap: 1em;
    }
